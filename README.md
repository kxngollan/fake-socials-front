<h1 align="center"> Fake Socials</h1>

# Built with

- NextJS + ReactJs
- Socket.Io

## Getting Started

First, Clone the repo:

```bash
#HTTPS
$ git clone https://github.com/kxngollan/fake-socials-client.git

#SSH
$ git clone git@github.com:kxngollan/fake-socials-client.git

Second, run the development server:


1. npm install

2. npm run dev


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

`
```

## Key Features

- Follows of other friends to get a customised feed.
- Ability to have live chat with other users.
- Uploading of post with images or videos.
- Customisation of profile including profile picture, links to other social media.
- Likes and comments for posts as well as reply to comments.
